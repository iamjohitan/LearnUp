🛠️ Instalación y uso
1. Clonar repositorio
git clone https://github.com/iamjohitan/LearnUp.git
cd LearnUp
2. Instalar dependencias

Backend:

cd backend
npm install

Frontend:

cd frontend
npm install
3. Ejecutar el proyecto

Backend:

npm run dev

Frontend:

npm start
